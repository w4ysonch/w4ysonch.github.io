# w4ysonch.github.io

Wayson的技术博客 — 专注 C/C++、RTOS、嵌入式Linux、AI CLI开发与学习笔记

## 技术栈

- **框架**: [Hexo](https://hexo.io/) 8.x
- **主题**: [Butterfly](https://butterfly.js.org/) 5.x
- **部署**: GitHub Pages + GitHub Actions

## 本地开发

```bash
npm install
npm run server     # 启动本地预览 http://localhost:4000
npm run build      # 生成静态文件到 public/
```
